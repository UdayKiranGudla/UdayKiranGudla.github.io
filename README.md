# Resume — Uday Kiran Gudla

This repository contains the resume of Uday Kiran Gudla. Open `index.html` in your browser to view the resume or print/save it as PDF.

Path: `index.html`

## Clone this repository

Make sure [Git](https://git-scm.com/downloads) is installed, then run:

```bash
git clone https://github.com/UdayKiranGudla/UdayKiranGudla.github.io.git
cd UdayKiranGudla.github.io
```

Then open `index.html` in your browser:

- **Windows:** `start index.html`
- **macOS:** `open index.html`
- **Linux:** `xdg-open index.html`

### Optional: clone with SSH

```bash
git clone git@github.com:UdayKiranGudla/UdayKiranGudla.github.io.git
```
