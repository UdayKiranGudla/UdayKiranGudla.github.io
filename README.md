This repository contains a single-page resume (`index.html`) optimized for screen and print (A4/PDF).

Preview locally

Open the file in your default browser from PowerShell:

```powershell
Set-Location 'f:\MyProfile\UdayKiranGudla.github.io'
Start-Process index.html
```

Save as PDF

- Click the **Download PDF** button on the page, or use the browser Print dialog and choose "Save as PDF".
- The page uses A4 print settings and hides the header/footer when printing.

Publish on GitHub Pages

1. Push this repository to GitHub (owner `UdayKiranGudla`).
2. In the repository Settings -> Pages, set Source to the `main` branch and the `/ (root)` folder.
3. After a minute, your site will be live at `https://UdayKiranGudla.github.io/`.

Customization

- Edit `index.html` to update contact info, summary, experience and projects.
- To change accent color, edit the `--accent` CSS variable in the `<style>` section.
